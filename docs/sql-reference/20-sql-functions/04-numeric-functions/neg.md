---
title: NEG
---

Negates a numeric value.

## Syntax

```sql
NEG( <x> )
```

## Aliases

- [MINUS](minus.md)
- [NEGATE](negate.md)
- [SUBTRACT](subtract.md)

## Examples

```sql
SELECT MINUS(PI()), NEG(PI()), NEGATE(PI()), SUBTRACT(PI());

┌───────────────────────────────────────────────────────────────────────────────────┐
│     minus(pi())    │      neg(pi())     │    negate(pi())    │   subtract(pi())   │
├────────────────────┼────────────────────┼────────────────────┼────────────────────┤
│ -3.141592653589793 │ -3.141592653589793 │ -3.141592653589793 │ -3.141592653589793 │
└───────────────────────────────────────────────────────────────────────────────────┘
```
