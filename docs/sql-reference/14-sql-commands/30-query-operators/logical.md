---
title: Logical Operators
---

| Operator | Description                            | Example       | Result |
|----------|----------------------------------------|---------------|--------|
| **AND**  | Matches both expressions (`a` and `b`) | **1 AND 1**   | TRUE   |
| **NOT**  | Does not match the expression          | **NOT 1**     | FALSE  |
| **OR**   | Matches either expression              | **1 OR 0**    | TRUE   |
