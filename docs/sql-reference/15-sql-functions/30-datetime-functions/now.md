---
title: NOW
---

Returns the current date and time.

## Syntax

```sql
NOW()
```

## Return Type

`TIMESTAMP`, returns date and time in “YYYY-MM-DD hh:mm:ss.ffffff” format.

## Examples

```sql
SELECT NOW();
+----------------------------+
| NOW()                      |
+----------------------------+
| 2022-10-17 08:13:58.751449 |
+----------------------------+
```
