---
title: REVERSE
---

Returns the string str with the order of the characters reversed.

## Syntax

```sql
REVERSE(<str>)
```

## Arguments

| Arguments | Description       |
|-----------|-------------------|
| `<str>`   | The string value. |

## Return Type

`VARCHAR`

## Examples

```sql
SELECT REVERSE('abc');
+----------------+
| REVERSE('abc') |
+----------------+
| cba            |
+----------------+
```
