---
title: H3_HEX_RING
---

Returns the "hollow" ring of hexagons at exactly grid distance `k` from the given [H3](https://eng.uber.com/h3/) index.

## Syntax

```sql
H3_HEX_RING(h3, k)
```

## Examples
