---
title: H3_EDGE_LENGTH_M
---

Returns the average hexagon edge length in meters at the given resolution. Excludes pentagons.

## Syntax

```sql
H3_EDGE_LENGTH_M(1)
```

## Examples

```sql
┌─────────────────────┐
│ h3_edge_length_m(1) │
├─────────────────────┤
│   483056.8390711111 │
└─────────────────────┘
```
