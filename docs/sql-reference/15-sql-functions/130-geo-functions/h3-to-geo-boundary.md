---
title: H3_TO_GEO_BOUNDARY
---

Returns an array containing the longitude and latitude coordinates of the vertices of the hexagon corresponding to the [H3](https://eng.uber.com/h3/) index. 

## Syntax

```sql
H3_TO_GEO_BOUNDARY(h3)
```

## Examples
