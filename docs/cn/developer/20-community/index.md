# 欢迎来到 Databend 社区！

加入我们，为 Databend 项目的成长贡献力量。参与方式如下：

- **[RFCs](/developer/community/rfcs)**: 分享您对提议的功能和更改的见解。

- **[开发路线图](01-roadmap.md)**: 了解 Databend 的下一步发展方向。

- **[贡献者指南](/developer/community/contributor)**: 了解如何为我们的成功做出贡献。
