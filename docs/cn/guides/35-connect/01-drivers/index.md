---
title: 驱动
---

使用官方驱动将您的应用程序连接到 Databend：

| 语言 | 驱动 | 使用场景 |
|-----|------|---------|
| [**Go**](/guides/connect/drivers/golang) | 原生驱动 | 后端服务、微服务 |
| [**Python**](/guides/connect/drivers/python) | Python 连接器 | 数据科学、分析、机器学习 |
| [**Node.js**](/guides/connect/drivers/nodejs) | JavaScript 驱动 | Web 应用 |
| [**Java**](/guides/connect/drivers/java) | JDBC 驱动 | 企业应用 |
| [**Rust**](/guides/connect/drivers/rust) | 原生驱动 | 系统编程 |
