将 Databend 连接到流行的 BI 和可视化工具：

| [**Metabase**](/guides/connect/visualization/metabase) | 自助式商业智能 |
| [**Jupyter**](/guides/connect/visualization/jupyter) | 交互式数据科学笔记本 |
| [**Deepnote**](/guides/connect/visualization/deepnote) | 协作数据笔记本 |
| [**MindsDB**](/guides/connect/visualization/mindsdb) | 机器学习平台 |
| [**Redash**](/guides/connect/visualization/redash) | SQL 驱动的仪表盘 |
