# Databend AI 和 ML

Databend 提供了两种 AI 和 ML 集成方法：

| 方法 | 特性 | 使用场景 |
|----------|----------|-----------|
| **[外部函数](01-external-functions.md)** ✓ *推荐* | • 自定义模型<br/>• GPU 部署<br/>• 自定义管道<br/>• 数据隐私 | • 专业领域<br/>• 高性能<br/>• 隐私要求 |
| **[内置函数](02-built-in-functions.md)** | • 文本补全<br/>• 嵌入<br/>• Vector 操作<br/>• 零设置 | • 快速原型设计<br/>• 通用 NLP<br/>• 简单实现 |
