欢迎来到 Databend（发音为 /ˈdeɪtəˌbɛnd/）<Speaker /> 文档。

**Databend** 是一个开源的、基于 Serverless 形态和存算分离架构的云原生数据湖仓 (Data Lakehouse)，构建在对象存储之上。它提供极致的性能和快速的弹性扩展能力，致力于成为开源版的 **Snowflake**。
