欢迎来到 Databend（发音为 /ˈdeɪtəˌbɛnd/）<Speaker /> 文档。

Databend 是云端多模态 Data+AI 数仓，原生列式存储覆盖结构化、半结构化、非结构化与向量数据，以对象存储为底座，数据写入即可使用。

它提供一体化的 Datastore、向量数据库、分析、搜索、地理空间引擎，并通过 SQL 统一对外服务，帮助团队集中管理数据、智能检索并实时获得洞察。
