## 获取支持

您可以通过多种渠道获取支持：

- **云控制台**：登录 Databend Cloud 控制台，在菜单选项中选择 **支持 → 创建新票**，以打开新的支持案件并查看提交案件的状态。
- **状态页面**：订阅我们的 [状态页面](https://status.databend.com) ，以快速了解影响我们平台的任何事件。
- **文档**：浏览我们的全面 [文档](https://docs.databend.com) ，获取指导、教程和参考资料。

## 支持服务级别

Databend Cloud 根据订阅级别提供不同的支持级别：

| 特性 | 个人版 | 商业版 | edicated 版 |
|------|--------|--------|-----------|
| 支持票的 logging 和跟踪 | ✓      | ✓      | ✓         |
| 24 小时、8 小时和 4 小时响应窗口针对严重性 1 问题 | ✓      | ✓      | ✓         |
| 非严重性 1 问题的响应 | ✓      | ✓      | ✓         |

### 严重性等级

- **严重性 1**：停止业务运营的严重问题，无 workaround
- **严重性 2**：重大功能受影响，性能显著下降
- **严重性 3**：部分功能失效，业务影响较小
- **严重性 4**：一般问题、建议或功能请求

:::note
请注意，只有商业版和 dedicated 版的客户有支持事件的 SLA。如果您使用个人版，我们承诺会尽力尽快回答您的问题，但建议您也探索我们的社区资源：

- [Databend 社区 Slack 频道](https://link.databend.com/join-slack)
- [GitHub 讨论](https://github.com/datafuselabs/databend/discussions)
:::

## 企业支持

对于需要 Mission-Critical 部署的客户，我们的 dedicated 版提供增强的支持选项，包括：

- 所有严重性等级的优先响应时间
- edicated 支持工程师
- 预警监控和问题解决
- 正常健康检查和优化建议

请联系我们了解我们企业支持 offerings 的更多信息。
