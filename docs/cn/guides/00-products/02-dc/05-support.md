## 获取支持

您可通过以下渠道获取支持：

- **云控制台**：登录 Databend Cloud 控制台，在菜单选项中选择 **支持 → 创建新工单** 以提交新支持工单，并查看已提交工单的状态。
- **状态页**：订阅我们的[状态页](https://status.databend.com)，及时获取影响平台运行的事件通知。
- **文档中心**：浏览全面的[文档中心](https://docs.databend.com)，查阅指南、教程和参考资料。

## 支持服务级别

Databend Cloud 根据订阅版本提供不同级别的支持：

| 功能 | 基础版 | 商业版 | 专属版 |
|---------|----------|----------|-----------|
| 支持工单记录与追踪 | ✓ | ✓ | ✓ |
| 严重性 1 级问题的 4/7 覆盖与响应窗口 | 24 小时 | 8 小时 | 4 小时 |
| 非严重性 1 级问题的响应时间 | 48 小时 | 24 小时 | 8 小时 |

### 问题严重性等级

- **严重性 1**：导致业务停滞且无临时解决方案的关键问题
- **严重性 2**：主要功能受影响且性能严重下降
- **严重性 3**：部分功能失效，对业务影响较小
- **严重性 4**：常规咨询、建议或功能请求

:::note
请注意，仅商业版和专属版客户享有支持事件服务级别协议 (SLA)。若您使用基础版——我们将尽力及时解答您的问题，同时建议您利用社区资源：

- [Databend 社区 Slack 频道](https://link.databend.com/join-slack)
- [GitHub 讨论区](https://github.com/datafuselabs/databend/discussions)
:::

## 企业级支持

针对关键业务部署场景，我们的专属版提供增强支持选项，包括：

- 所有严重性等级的优先响应
- 专属支持工程师
- 主动监控与问题解决
- 定期健康检查及优化建议

联系[销售团队](https://www.databend.com/contact-us/)了解企业级支持的更多详情。
