欢迎来到 Databend 指南，涵盖了 Databend 版本的所有方面。

## 术语表

- **`Databend`** 指的是[开源版本](https://github.com/datafuselabs/databend)。
- **`Databend Cloud`** 是一个完全[托管的云服务](https://databend.com)。
- **`Databend(Cloud)`** 统称 `Databend` 和 `Databend Cloud`。
