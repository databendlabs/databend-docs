# 术语表

- **`Databend`** 指的是 [开源版本](https://github.com/datafuselabs/databend) 。
- **`Databend Cloud`** 是一个[完全托管的云服务](https://databend.com) 。
- **`Databend(Cloud)`** 同时适用于 `Databend` 和 `Databend Cloud` 。
