:::note
Databend Cloud 计划扩展对其他云平台和地区的支持。如果您的项目需要特定云平台或地区的支持，请点击 Databend Cloud 首页右上角的 **联系销售** 按钮，以与 Databend Cloud 团队取得联系。
:::

当您注册 Databend Cloud 账户时，您需要选择一个云平台和地区。一旦账户成功创建，所选的云平台和地区将无法更改。
