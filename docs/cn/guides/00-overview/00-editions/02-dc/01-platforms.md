:::note
Databend Cloud 计划扩展对更多云平台和区域的支持。如果您的项目需要特定云平台或区域的支持，请点击 Databend Cloud 主页右上角的 **联系销售** 按钮，与 Databend Cloud 团队取得联系。
:::

您在开通 Databend Cloud 账号时需要选定一个云厂商及地域。开通成功后，选定的云厂商及地域暂不支持修改。
