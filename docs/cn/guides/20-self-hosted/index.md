# Databend 私有部署

[Databend](https://github.com/databendlabs/databend) 开源云数仓，Rust 打造，部署在您自己的环境里。

## 一览

| | |
|---|---|
| 🦀 **Rust 内核** | 安全高效，性能卓越 | 
| 🔓 **开源免费** | Apache 2.0，社区活跃 |
| 🏢 **完全掌控** | 数据不出门，环境随心配 |
| ☁️ **云原生** | 对象存储优先架构 |

## 选择版本

| 版本 | 适合 |
|------|------|
| [**社区版**](self-hosted/editions/community) | 开发测试、中小规模 |
| [**企业版**](self-hosted/editions/enterprise/) | 规模化生产、企业特性 |
