# Databend Cloud

基于开源 [Databend](https://github.com/databendlabs/databend) 内核打造的全托管云数仓——不是简单的托管部署，而是从架构层重新设计的多租户 Serverless 云原生平台。

## 一览

| | |
|---|---|
| ⚡ **< 500ms** 冷启动 | 🔐 **SOC 2 Type II** 认证 |
| 💰 **> 50%** 成本节省（对比 Snowflake） | 📊 **99.95%** 可用性 SLA |

## 核心优势

| | |
|---|---|
| **Serverless** | 无需管理集群，空闲自动释放 |
| **存算分离** | 独立扩展，冷热分层优化成本 |
| **企业级安全** | RBAC、数据脱敏、加密、PrivateLink、审计日志 |
| **Snowflake 兼容** | 熟悉的 SQL 语法，平滑迁移 |
