# 欢迎加入 Databend 社区！

加入我们，共同推动 Databend 项目的发展。以下是您可以参与的方式：

- **[RFCs](/guides/overview/community/rfcs)**：分享您对提议功能和更改的见解。

- **[开发路线图](01-roadmap.md)**：了解 Databend 的下一步计划。
