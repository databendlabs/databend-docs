## v0.7 - 2022年3月28日

**新特性**

- [Query] 增加 DateTime64 数据类型 (#3338)
- [Query] 增加半结构化 variant 数据类型 (#4348)
- [Query] 增加 `group by` 可空列 (#4079)
- [Query] 增加 `read datasource` 的 filter 下推 (#3512)
- [Query] 增加函数单调性检查 (#3008)
- [Query] 增加对 Java/Python/Golang 客户端和 JDBC 驱动的支持
- [Query] 增加对 github 外部数据源的支持 (#2975)
- [Query] 增加对 `CREATE TABLE [IF NOT EXISTS] [db.]table LIKE [db.]origin_table AS SELECT query` 的支持 (#2457)
- [Query] 增加对 `Copy INTO <table> from external location` 的支持 (#3586 #4373)
- [Query] 增加对 `COPY INTO <table> FROM internal/external stage` 的支持 (#4437)
- [Query] 增加对 `ALTER TABLE [ IF EXISTS ] <name> RENAME TO <new_table_name>` 的支持 (#4532)
- [Query] 增加对 `[NOT] IN (a, b, c, ...) InList` 的支持 (#3805)
- [Query] 增加用户自定义函数 (#3440)
- [Query] 增加用于多租户查询统计的 `system.query_log` (#2978)
- [Query] 增加用于管理操作的 `CALL` 存储过程 (#4267)
- [Query] 增加对 CSV/Parquet/Json 文件流式加载的支持 (#4496 #4561)
- [Query] 增加将 markdown 文档嵌入到 system.functions 的支持 (#4552)
- [Query] 增加对 json, yaml, toml 格式配置文件的支持 (#4517)
- [Query] 增加对指定表达式作为列默认值的支持 (#3229)
- [User] 增加对基本 RBAC 权限的支持 (#2793)
- [Storage] 增加 database_id/table_id 到数据文件前缀 (#4242)
- [Storage] 增加版本到存储布局以进行升级 (#4243)
- [Meta] 增加对多租户的支持 (#3759)
- [Meta] 使用 gRPC 替换 arrow flight service (#3605)

**改进**

- [Query] 将基于拉取改为新的基于拉取和推送的处理器
- [Query] 提高并行写入性能
- [Doc] 更好的文档 https://docs.databend.com
