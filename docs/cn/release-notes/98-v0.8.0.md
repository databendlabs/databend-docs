**发布日期：2022 年 8 月 18 日**

此版本的 Databend 包含新功能和改进。

### 新功能和改进

- Query(Copy): 使 copy into 可以并行加载目录中的多个文件 #4584
- Query(Copy): 功能：支持 COPY 命令的 JSON 文件格式 #4527
- Query(Copy): 将数据卸载到 Stage #4498
- Query(Copy): 提高从外部位置 COPY 的性能 #4308
- Query(Insert): 功能：异步插入 #4577
- Query(View): 功能：支持 View Create/Select/Alter #4610
- Query(Insert): 功能：提高 insert into 的性能 #4413
- Query(SQL): 将 sqlparser-rs 更新到最新版本以支持 Unsigned\* #4877
- Query (TimeZone): 功能：适应不同国家/地区，支持用户更改时区 #4651
- Query(DataType): 支持半结构化数据类型（如 json 格式）#3916
- Query(information_schema): 支持 information_schema 数据库 #4659
- Query(DataType): Datetime 类型尝试与 chrono 相乘时发生溢出 #4874
- Query(ALTER): 支持：ALTER DATABASE [ IF EXISTS ] db_name RENAME TO new_db_name #4839
- Query(DML): 支持查询：show table status #4757
- Query(DML) ISSUE-4459: 添加对 alter table rename 语句的支持 #4532
- Query(DataType): 支持 ArrayT 数据类型
- Query(Function): feat: 添加标量函数 humanize #5073
- Query(Function): feat(functions): 支持聚合函数 retention #4970
- Query(Function): feature: 支持 REGEXP_REPLACE 函数 #4944
- Query(Function): feature: 支持 REGEXP_SUBSTR 函数 #4771
- Query(Function): feat(function): 支持半结构化函数 GET/GET_IGNORE_CASE/GET_PATH #4684
- Query(Function) : 支持 PARSE_JSON / TRY_PARSE_JSON 函数 #4534
- Query(Source): feat(source): 为流式加载添加 ndjson 源 #4561
- Query(Settings): feat(setting): 支持全局设置 #6579
- Query(Handler): feat(query): 弃用 clickhouse 的 tcp 协议支持 #7012
- Query(Planner): 支持 CTE(Common Table Expression) #6246
- Query(UDF): SQL UDFs #3994
- Query(GroupBy): 两级哈希表优化 #4602
- Query(SQL): [SQL] 支持 Common Table Expressions #514
- Query(Optimizer): 优化器 rule_push_down_filter_scan 未启用 #7069

- Storage(Delete): Feat: statement delete from... #5691
- Storage(Azblob): feat: 实现 azblob 支持 #5025
- Storage(TimeTravel): Feat : undrop table & show history #5562
- Storage(clustering_information): feat(fuse): 添加 system$clustering_information 函数 #5426
- Storage(Index): feat: 启用 bloom filter index #6639
- Storage(ClusterKey): Feature: Table Cluster Key #4268
- Storage(COPY): 支持在 COPY 语句中读取压缩文件 #5380
- Storage(External): Hive 作为外部数据源 #4826
- Storage(predict): refactor(query): 谓词下推支持多个表达式 #7078

- API(REST): Feature: 添加 ClickHouse REST 处理程序 #4568
- API(JDBC): 建议：支持 DBeaver 连接 Databend #4581

- Meta: Feature: [metasrv]: 添加 API watch #4393
- Meta: 添加 API 以从 metasrv 集群中删除节点 #4312

- Cloud(User): 数据库和表上的细粒度用户权限 #2894
- Executor(Processor): Feature: 支持中止 insert 查询 #4646
- Planner: 迁移到新的 planner 框架 #3747
- OpenDAL: Native compress support opendal#70

### 下载和安装 Databend

Databend 可在各种 Linux 平台和 macOS 上使用。单击[此处](https://github.com/databendlabs/databend/releases/tag/v0.8.0-nightly)以在 GitHub 上打开发布页面，然后选择并下载适用于您的平台的安装程序。

要部署 Databend，请按照[部署指南](/guides/deploy)进行操作。

### 加载数据

有很多方法可以将数据加载到 Databend 中。您可以根据数据的存储位置使用命令行界面 (CLI)、API 或 mysqldump 客户端实用程序。有关详细信息，请参阅[加载数据](/guides/load-data/load)教程。
