**发布日期：2023 年 1 月 13 日，星期五**

Databend v0.9.0 于 2023 年 1 月 13 日正式发布！与 v0.8.0 相比，我们在 v0.9.0 中进行了 5,000 多次提交、700 多项优化和修复、4,347 个文件更改以及 340,000 行代码更改。感谢所有参与的社区伙伴，感谢所有让 Databend 变得更好的人！

**完整变更日志**：https://github.com/databendlabs/databend/compare/v0.8.0-nightly...v0.9.0-nightly

## 主要特性 :sparkles:

- 全新的类型系统
- 实现 Rust 版本的 JSONB
- 完全支持 JOIN
- 支持基于成本的优化器
- 具有 [strawboat](https://github.com/sundy-li/strawboat) 的原生存储格式
- 高效的布隆过滤器过滤
- 设计并开源 Serverless DataSharing 协议
- Stage

## 重要改进 :tada:

- `read_parquet` 支持直接读取本地 parquet 文件，无需导入
- 常用函数性能优化，常用 GEO 函数支持
- Distinct 性能优化
- 自适应字符串哈希表
- SQLancer 集成
- Parquet 读取加速
- 之前的 python 版本 sqllogictest 已被重写为使用 Rust
- 支持 NDJSON 和 JSON 输出格式
- ALTER TABLE 支持 recluster
- 支持 hyperloglog 更新和删除：https://db.in.tum.de/~freitag/papers/p23-freitag-cidr19.pdf
