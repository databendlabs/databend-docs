---
title: SIPHASH64
---

生成一个64位的[SipHash](https://en.wikipedia.org/wiki/SipHash)哈希值。

## 语法

```sql
SIPHASH64(<expr>)
```

## 别名

- [SIPHASH](siphash.md)

## 示例

```sql
SELECT SIPHASH('1234567890'), SIPHASH64('1234567890');

┌─────────────────────────────────────────────────┐
│ siphash('1234567890') │ siphash64('1234567890') │
├───────────────────────┼─────────────────────────┤
│  18110648197875983073 │    18110648197875983073 │
└─────────────────────────────────────────────────┘
```
