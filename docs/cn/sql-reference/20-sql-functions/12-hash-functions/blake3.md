---
title: BLAKE3
---

计算字符串的 BLAKE3 256 位校验和。返回值为 64 个十六进制字符的字符串，如果参数为 NULL，则返回 NULL。

## 语法

```sql
BLAKE3(<expr>)
```

## 示例

```sql
SELECT BLAKE3('1234567890');

┌──────────────────────────────────────────────────────────────────┐
│                       blake3('1234567890')                       │
├──────────────────────────────────────────────────────────────────┤
│ d12e417e04494572b561ba2c12c3d7f9e5107c4747e27b9a8a54f8480c63e841 │
└──────────────────────────────────────────────────────────────────┘
```
