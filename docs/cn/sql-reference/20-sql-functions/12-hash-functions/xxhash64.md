---
title: XXHASH64
---

计算字符串的 xxHash64 64 位哈希值。该值以 UInt64 类型返回，如果参数为 NULL，则返回 NULL。

## 语法

```sql
XXHASH64(expr)
```

## 示例

```sql
SELECT XXHASH64('1234567890');

┌────────────────────────┐
│ xxhash64('1234567890') │
├────────────────────────┤
│   12237639266330420150 │
└────────────────────────┘
```
