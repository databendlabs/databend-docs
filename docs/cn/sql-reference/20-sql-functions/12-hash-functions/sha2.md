---
title: SHA2
---

计算 SHA-2 系列哈希函数（SHA-224、SHA-256、SHA-384 和 SHA-512）。如果哈希长度不是允许的值之一，则返回值为 NULL。否则，函数结果是一个包含所需位数的十六进制数字字符串的哈希值。

## 语法

```sql
SHA2(<expr>, <expr>)
```

## 示例

```sql
SELECT SHA2('1234567890', 0);

┌──────────────────────────────────────────────────────────────────┐
│                       sha2('1234567890', 0)                      │
├──────────────────────────────────────────────────────────────────┤
│ c775e7b757ede630cd0aa1113bd102661ab38829ca52a6422ab782862f268646 │
└──────────────────────────────────────────────────────────────────┘
```
