---
title: MD5
---

计算字符串的 MD5 128 位校验和。返回值是一个 32 位的十六进制字符串，如果参数为 NULL，则返回 NULL。

## 语法

```sql
MD5(<expr>)
```

## 示例

```sql
SELECT MD5('1234567890');

┌──────────────────────────────────┐
│         md5('1234567890')        │
├──────────────────────────────────┤
│ e807f1fcf82d132f9bb018ca6738a19f │
└──────────────────────────────────┘
```
