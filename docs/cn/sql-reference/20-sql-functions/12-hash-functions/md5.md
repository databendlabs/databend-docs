---
title: MD5
---

计算字符串的 MD5 128 位校验和。该值作为 32 位十六进制数字的字符串返回，如果参数为 NULL，则返回 NULL。

## 句法

```sql
MD5(<expr>)
```

## 示例

```sql
SELECT MD5('1234567890');

┌──────────────────────────────────┐
│         md5('1234567890')        │
├──────────────────────────────────┤
│ e807f1fcf82d132f9bb018ca6738a19f │
└──────────────────────────────────┘
```
