---
title: SLEEP
---

在每个数据块上休眠 `seconds` 秒。

!!! warning 
    仅用于需要休眠的测试场景。


## 语法

```sql
SLEEP(seconds)
```

## 参数

| 参数        | 描述         |
| ----------- | ------------ |
| seconds     | 必须是一个非负数的常量列，可以是任何非负数或浮点数。 |

## 返回类型

UInt8

## 示例

```sql
SELECT sleep(2);
+----------+
| sleep(2) |
+----------+
|        0 |
+----------+
```
