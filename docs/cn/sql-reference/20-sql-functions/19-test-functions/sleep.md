---
title: SLEEP
---

使查询在处理每个数据块时休眠 `seconds` 秒。

:::caution
主要用于测试目的。
:::

## 语法

```sql
SLEEP(seconds)
```

## 参数

| 参数        | 描述         |
| ----------- | ------------ |
| seconds     | 指定休眠时间的非负常数（整数或浮点数）。 |

## 返回类型

UInt8

## 示例

```sql
SELECT sleep(2);
+----------+
| sleep(2) |
+----------+
|        0 |
+----------+
```
