---
title: TYPEOF
---

TYPEOF 函数用于返回数据类型的名称。

## 语法

```sql
TYPEOF( <expr> )
```

## 参数

| 参数        | 描述        |
| ----------- | ----------- |
| `<expr>` | 任何表达式。<br />这可以是列名、另一个函数的结果或数学运算。

## 返回类型

字符串

## 示例

```sql
SELECT typeof(1::INT);
+------------------+
| typeof(1::Int32) |
+------------------+
| INT              |
+------------------+
```
