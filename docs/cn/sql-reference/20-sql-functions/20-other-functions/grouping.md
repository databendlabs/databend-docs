---
title: GROUPING 
---

返回一个位掩码，指示当前分组集中不包含哪些 `GROUP BY` 表达式。位从最右边的参数开始分配，对应于最低有效位；如果当前结果行生成的分组集的分组条件中包含相应的表达式，则每一位为 0，如果不包含，则为 1。

## 语法

```sql
GROUPING ( expr [, expr, ...] )
```

:::note
`GROUPING` 只能与 `GROUPING SETS`、`ROLLUP` 或 `CUBE` 一起使用，并且其参数必须在分组集列表中。
:::

## 参数

分组集项目。

## 返回类型

UInt32.

## 示例

```sql
select a, b, grouping(a), grouping(b), grouping(a,b), grouping(b,a) from t group by grouping sets ((a,b),(a),(b), ()) ;
+------+------+-------------+-------------+----------------+----------------+
| a    | b    | grouping(a) | grouping(b) | grouping(a, b) | grouping(b, a) |
+------+------+-------------+-------------+----------------+----------------+
| NULL | A    |           1 |           0 |              2 |              1 |
| a    | NULL |           0 |           1 |              1 |              2 |
| b    | A    |           0 |           0 |              0 |              0 |
| NULL | NULL |           1 |           1 |              3 |              3 |
| a    | A    |           0 |           0 |              0 |              0 |
| b    | B    |           0 |           0 |              0 |              0 |
| b    | NULL |           0 |           1 |              1 |              2 |
| a    | B    |           0 |           0 |              0 |              0 |
| NULL | B    |           1 |           0 |              2 |              1 |
+------+------+-------------+-------------+----------------+----------------+
```
