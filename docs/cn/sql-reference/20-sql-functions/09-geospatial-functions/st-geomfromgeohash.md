返回一个 GEOMETRY 对象，该对象表示 [geohash](https://en.wikipedia.org/wiki/Geohash) 边界的多边形。

## 语法

```sql
ST_GEOMFROMGEOHASH(<geohash>)
```

## 参数

| 参数   | 说明                     |
|-------------|---------------------------------|
| `<geohash>` | 参数必须是 geohash。 |

## 返回类型

Geometry。

## 示例

```sql
SELECT
  ST_GEOMFROMGEOHASH(
    '9q60y60rhs'
  ) AS pipeline_geometry;
```
