返回表示 [geohash](https://en.wikipedia.org/wiki/Geohash) 边界多边形的 GEOGRAPHY 对象。

## 语法

```sql
ST_GEOGFROMGEOHASH(<geohash>)
```

## 参数

| 参数        | 说明                    |
|-------------|-------------------------|
| `<geohash>` | geohash 字符串。 |

## 返回类型

Geography。

## 示例

```sql
SELECT
  ST_ASWKT(
    ST_GEOGFROMGEOHASH(
      '9q60y60rhs'
    )
  ) AS pipeline_geography;
```
