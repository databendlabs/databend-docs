---
title: H3_HEX_RING
---

返回与给定 [H3](https://eng.uber.com/h3/) 索引的网格距离恰好为 `k` 的“空心”六边形环。

## 语法

```sql
H3_HEX_RING(h3, k)
```

## 示例

```sql
SELECT H3_HEX_RING(599686042433355775, 2);
```
