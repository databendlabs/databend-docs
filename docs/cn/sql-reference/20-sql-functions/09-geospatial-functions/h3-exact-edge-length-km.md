---
title: H3_EXACT_EDGE_LENGTH_KM
---

计算此有向边（directed edge）的长度，单位为公里。

## 语法

```sql
H3_EXACT_EDGE_LENGTH_KM(h3)
```

## 示例

```sql
SELECT H3_EXACT_EDGE_LENGTH_KM(1319695429381652479);

┌──────────────────────────────────────────────┐
│ h3_exact_edge_length_km(1319695429381652479) │
├──────────────────────────────────────────────┤
│                            8.267326832647143 │
└──────────────────────────────────────────────┘
```
