---
title: H3_GET_FACES
---

查找与给定 [H3](https://eng.uber.com/h3/) 索引相交的所有二十面体面（Icosahedron Faces）。面以 0–19 之间的整数表示。

## 语法

```sql
H3_GET_FACES(h3)
```

## 示例

```sql
SELECT H3_GET_FACES(599119489002373119);

┌──────────────────────────────────┐
│ h3_get_faces(599119489002373119) │
├──────────────────────────────────┤
│ [0,1,2,3,4]                      │
└──────────────────────────────────┘
```
