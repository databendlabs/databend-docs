---
title: GEO_DISTANCE
---

返回地球上两点之间的近似距离（米）。点使用经纬度（度）表示，距离基于 WGS84 近似计算。

## 语法

```sql
GEO_DISTANCE(<lon1>, <lat1>, <lon2>, <lat2>)
```

## 参数

| 参数 | 描述 |
|-----------|-------------|
| `<lon1>` | 第一个点的经度（度）。 |
| `<lat1>` | 第一个点的纬度（度）。 |
| `<lon2>` | 第二个点的经度（度）。 |
| `<lat2>` | 第二个点的纬度（度）。 |

## 返回类型

Float32。

## 示例

```sql
SELECT GEO_DISTANCE(55.755831, 37.617673, -55.755831, -37.617673) AS distance;

╭────────────╮
│  distance  │
├────────────┤
│ 14128353.0 │
╰────────────╯
```
