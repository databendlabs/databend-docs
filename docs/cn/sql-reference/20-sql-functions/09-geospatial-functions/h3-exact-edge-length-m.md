---
title: H3_EXACT_EDGE_LENGTH_M
---

计算此有向边（Directed Edge）的长度，单位为米。

## 语法

```sql
H3_EXACT_EDGE_LENGTH_M(h3)
```

## 示例

```sql
SELECT H3_EXACT_EDGE_LENGTH_M(1319695429381652479);

┌─────────────────────────────────────────────┐
│ h3_exact_edge_length_m(1319695429381652479) │
├─────────────────────────────────────────────┤
│                           8267.326832647143 │
└─────────────────────────────────────────────┘
```
