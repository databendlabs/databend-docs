---
title: GREAT_CIRCLE_DISTANCE
---

返回球面上两点之间的大圆距离（米）。点使用经纬度（度）表示。

## 语法

```sql
GREAT_CIRCLE_DISTANCE(<lon1>, <lat1>, <lon2>, <lat2>)
```

## 参数

| 参数 | 描述 |
|-----------|-------------|
| `<lon1>` | 第一个点的经度（度）。 |
| `<lat1>` | 第一个点的纬度（度）。 |
| `<lon2>` | 第二个点的经度（度）。 |
| `<lat2>` | 第二个点的纬度（度）。 |

## 返回类型

Float32。

## 示例

```sql
SELECT GREAT_CIRCLE_DISTANCE(55.755831, 37.617673, -55.755831, -37.617673) AS distance;

╭────────────╮
│  distance  │
├────────────┤
│ 14128353.0 │
╰────────────╯
```
