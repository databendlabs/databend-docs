---
title: POINT_IN_POLYGON
---

计算给定点是否位于由多个点依次连接形成的多边形内部。多边形是按坐标对出现顺序连接而成的封闭图形，改变坐标对的顺序可能得到不同形状。

## 语法

```sql
POINT_IN_POLYGON((x,y), [(a,b), (c,d), (e,f) ... ])
```

## 示例

```sql
SELECT POINT_IN_POLYGON((3., 3.), [(6, 0), (8, 4), (5, 8), (0, 2)]);

┌────────────────────────────────────────────────────────────┐
│ point_in_polygon((3, 3), [(6, 0), (8, 4), (5, 8), (0, 2)]) │
├────────────────────────────────────────────────────────────┤
│                                                          1 │
└────────────────────────────────────────────────────────────┘
```
