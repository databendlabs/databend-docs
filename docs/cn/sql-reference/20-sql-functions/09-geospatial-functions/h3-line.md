---
title: H3_LINE
---

返回给定两个 [H3](https://eng.uber.com/h3/) 索引（Index）之间的索引连线。

## 语法

```sql
H3_LINE(h3, a_h3)
```

## 示例

```sql
SELECT H3_LINE(599119489002373119, 599119491149856767);

┌─────────────────────────────────────────────────┐
│ h3_line(599119489002373119, 599119491149856767) │
├─────────────────────────────────────────────────┤
│ [599119489002373119,599119491149856767]         │
└─────────────────────────────────────────────────┘
```
