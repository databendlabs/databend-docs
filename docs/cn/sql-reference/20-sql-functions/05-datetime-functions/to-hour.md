---
title: TO_HOUR
---

将包含时间的日期（timestamp/datetime）转换为一个 UInt8 数字，其中包含 24 小时制的小时数（0-23）。
此函数假定如果时钟提前，则提前一小时，发生在凌晨 2 点；如果时钟倒退，则倒退一小时，发生在凌晨 3 点（这并不总是正确的——即使在莫斯科，时钟也有两次在不同的时间更改）。

## 语法

```sql
TO_HOUR(<expr>)
```

## 参数

| 参数 | 描述 |
|-----------|-------------|
| `<expr>`  | timestamp   |

## 返回类型

`TINYINT`

## 示例

```sql
SELECT
    to_hour('2023-11-12 09:38:18.165575');

┌───────────────────────────────────────┐
│ to_hour('2023-11-12 09:38:18.165575') │
├───────────────────────────────────────┤
│                                     9 │
└───────────────────────────────────────┘
```
