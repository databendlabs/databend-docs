---
title: TIME_SLOT
---

将时间四舍五入到半小时。
## 句法

```sql
time_slot(<expr>)
```

## 参数

| 参数   | 描述 |
| ----------- | ----------- |
| `<expr>`    | timestamp   |

## 返回类型

`TIMESTAMP`，以“YYYY-MM-DD hh:mm:ss.ffffff”格式返回。

## 示例

```sql
SELECT
  time_slot('2023-11-12 09:38:18.165575');

┌─────────────────────────────────────────┐
│ time_slot('2023-11-12 09:38:18.165575') │
├─────────────────────────────────────────┤
│ 2023-11-12 09:30:00                     │
└─────────────────────────────────────────┘
```
