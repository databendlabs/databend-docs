---
title: TO_MINUTE
---

将带有时间的日期（时间戳/日期时间）转换为包含小时分钟数的 UInt8 数字（0-59）。

## 语法

```sql
TO_MINUTE(<expr>)
```

## 参数

| 参数       | 描述     |
|-----------|----------|
| `<expr>`  | 时间戳   |

## 返回类型

 `TINYINT`

## 示例

```sql
SELECT
    to_minute('2023-11-12 09:38:18.165575');

┌─────────────────────────────────────────┐
│ to_minute('2023-11-12 09:38:18.165575') │
├─────────────────────────────────────────┤
│                                      38 │
└─────────────────────────────────────────┘
```
