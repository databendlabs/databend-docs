---
title: TO_MINUTE
---

将带有时间（timestamp/datetime）的日期转换为一个 UInt8 数字，其中包含小时的分钟数（0-59）。

## 语法

```sql
TO_MINUTE(<expr>)
```

## 参数

| 参数 | 描述 |
|-----------|-------------|
| `<expr>`  | timestamp   |

## 返回类型

 `TINYINT`

## 示例

```sql
SELECT
    to_minute('2023-11-12 09:38:18.165575');

┌─────────────────────────────────────────┐
│ to_minute('2023-11-12 09:38:18.165575') │
├─────────────────────────────────────────┤
│                                      38 │
└─────────────────────────────────────────┘
```
