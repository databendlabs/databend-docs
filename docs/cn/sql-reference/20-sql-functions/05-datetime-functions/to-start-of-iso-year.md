---
title: TO_START_OF_ISO_YEAR
---

返回日期或带时间（时间戳/日期时间）的日期的 ISO 年的第一天。

## 语法

```sql
TO_START_OF_ISO_YEAR(<expr>)
```

## 参数

| 参数      | 描述           |
|-----------|----------------|
| `<expr>`  | date/timestamp |

## 返回类型

`DATE`，返回 “YYYY-MM-DD” 格式的日期。

## 示例

```sql
SELECT
  to_start_of_iso_year('2023-11-12 09:38:18.165575');

┌────────────────────────────────────────────────────┐
│ to_start_of_iso_year('2023-11-12 09:38:18.165575') │
├────────────────────────────────────────────────────┤
│ 2023-01-02                                         │
└────────────────────────────────────────────────────┘
```
