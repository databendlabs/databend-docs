---
title: TO_START_OF_TEN_MINUTES
---

将带时间（时间戳/日期时间）的日期向下舍入到十分钟间隔的开始。

## 句法

```sql
TO_START_OF_TEN_MINUTES(<expr>)
```

## 参数

| 参数 | 描述 |
|-----------|-------------|
| `<expr>`  | timestamp   |

## 返回类型

`TIMESTAMP`，以“YYYY-MM-DD hh:mm:ss.ffffff”格式返回日期。

## 示例

```sql
SELECT
  to_start_of_ten_minutes('2023-11-12 09:38:18.165575');

┌───────────────────────────────────────────────────────┐
│ to_start_of_ten_minutes('2023-11-12 09:38:18.165575') │
├───────────────────────────────────────────────────────┤
│ 2023-11-12 09:30:00                                   │
└───────────────────────────────────────────────────────┘
```
