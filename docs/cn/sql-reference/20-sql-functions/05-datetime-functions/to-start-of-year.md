---
title: TO_START_OF_YEAR
---

返回日期或日期时间（时间戳/日期时间）所在年份的第一天。

## 语法

```sql
TO_START_OF_YEAR(<expr>)
```

## 参数

| 参数       | 描述           |
|------------|----------------|
| `<expr>`   | 日期/时间戳    |

## 返回类型

`DATE`，返回格式为“YYYY-MM-DD”的日期。

## 示例

```sql
SELECT
  to_start_of_year('2023-11-12 09:38:18.165575');

┌────────────────────────────────────────────────┐
│ to_start_of_year('2023-11-12 09:38:18.165575') │
│                      Date                      │
├────────────────────────────────────────────────┤
│ 2023-01-01                                     │
└────────────────────────────────────────────────┘
```
