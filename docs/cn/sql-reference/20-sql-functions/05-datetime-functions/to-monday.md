---
title: TO_MONDAY
---

将日期或带时间的日期（timestamp/datetime）向下舍入到最近的星期一。
返回日期。

## 句法

```sql
TO_MONDAY(<expr>)
```

## 参数

| 参数   | 描述 |
| ----------- | ----------- |
| `<expr>` | date/timestamp |

## 返回类型

`DATE`，返回 “YYYY-MM-DD” 格式的日期。

## 示例

```sql
SELECT
    to_monday('2023-11-12 09:38:18.165575');

┌─────────────────────────────────────────┐
│ to_monday('2023-11-12 09:38:18.165575') │
├─────────────────────────────────────────┤
│ 2023-11-06                              │
└─────────────────────────────────────────┘
```
