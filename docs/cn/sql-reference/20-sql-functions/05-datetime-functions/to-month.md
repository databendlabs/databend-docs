---
title: TO_MONTH
---

将日期或带时间的日期（时间戳/日期时间）转换为包含月份编号（1-12）的 UInt8 数字。

## 语法

```sql
TO_MONTH(<expr>)
```

## 参数

| 参数       | 描述         |
|-----------|--------------|
| `<expr>`  | 日期/时间戳  |

## 别名

- [MONTH](month.md)

## 返回类型

 `TINYINT`

## 示例

```sql
SELECT NOW(), TO_MONTH(NOW()), MONTH(NOW());

┌─────────────────────────────────────────────────────────────┐
│            now()           │ to_month(now()) │ month(now()) │
├────────────────────────────┼─────────────────┼──────────────┤
│ 2024-03-14 23:34:02.161291 │               3 │            3 │
└─────────────────────────────────────────────────────────────┘
```
