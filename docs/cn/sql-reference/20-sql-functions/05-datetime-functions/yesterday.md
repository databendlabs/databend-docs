---
title: 昨日
---

返回昨天的日期，等同于 `today() - 1`。

## 语法

```sql
YESTERDAY()
```

## 返回类型

`DATE`，返回日期格式为“YYYY-MM-DD”。

## 示例

```sql
SELECT YESTERDAY(), TODAY()-1;

┌───────────────────────────┐
│ yesterday() │ today() - 1 │
├─────────────┼─────────────┤
│ 2024-05-21  │ 2024-05-21  │
└───────────────────────────┘
```
