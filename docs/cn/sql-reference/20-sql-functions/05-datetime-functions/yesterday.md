title: YESTERDAY
---

返回昨天的日期，与 `today() - 1` 相同。

## 语法

```sql
YESTERDAY()
```

## 返回类型

`DATE`，返回 “YYYY-MM-DD” 格式的日期。

## 示例

```sql
SELECT YESTERDAY(), TODAY()-1;

┌───────────────────────────┐
│ yesterday() │ today() - 1 │
├─────────────┼─────────────┤
│ 2024-05-21  │ 2024-05-21  │
└───────────────────────────┘
```
