---
title: TO_START_OF_FIVE_MINUTES
---

将日期和时间（时间戳/日期时间）向下舍入到五分钟间隔的开始。

## 语法

```sql
TO_START_OF_FIVE_MINUTES(<expr>)
```

## 参数

| 参数       | 描述       |
|------------|------------|
| `<expr>`   | 时间戳     |

## 返回类型

`TIMESTAMP`，返回日期格式为“YYYY-MM-DD hh:mm:ss.ffffff”。

## 示例

```sql
SELECT
  to_start_of_five_minutes('2023-11-12 09:38:18.165575')

┌────────────────────────────────────────────────────────┐
│ to_start_of_five_minutes('2023-11-12 09:38:18.165575') │
├────────────────────────────────────────────────────────┤
│ 2023-11-12 09:35:00                                    │
└────────────────────────────────────────────────────────┘
```
