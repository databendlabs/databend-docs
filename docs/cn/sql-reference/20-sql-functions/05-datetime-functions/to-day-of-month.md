---
title: TO_DAY_OF_MONTH
---

将日期或带时间（时间戳/日期时间）的日期转换为包含月份中天数（1-31）的 UInt8 数字。

## 句法

```sql
TO_DAY_OF_MONTH(<expr>)
```

## 参数

| 参数      | 描述           |
|-----------|----------------|
| `<expr>`  | 日期/时间戳    |

## 别名

- [DAY](day.md)

## 返回类型

`TINYINT`

## 示例

```sql
SELECT NOW(), TO_DAY_OF_MONTH(NOW()), DAY(NOW());

┌──────────────────────────────────────────────────────────────────┐
│            now()           │ to_day_of_month(now()) │ day(now()) │
├────────────────────────────┼────────────────────────┼────────────┤
│ 2024-03-14 23:35:41.947962 │                     14 │         14 │
└──────────────────────────────────────────────────────────────────┘
```
