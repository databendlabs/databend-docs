---
title: TO_START_OF_HOUR
---

将带时间（时间戳/日期时间）的日期向下舍入到小时的开始。
## 语法

```sql
TO_START_OF_HOUR(<expr>)
```

## 参数

| 参数 | 描述 |
|-----------|-------------|
| `<expr>`  | timestamp   |

## 返回类型

`TIMESTAMP`，以“YYYY-MM-DD hh:mm:ss.ffffff”格式返回日期。

## 示例

```sql
SELECT
  to_start_of_hour('2023-11-12 09:38:18.165575');

┌────────────────────────────────────────────────┐
│ to_start_of_hour('2023-11-12 09:38:18.165575') │
├────────────────────────────────────────────────┤
│ 2023-11-12 09:00:00                            │
└────────────────────────────────────────────────┘
```
