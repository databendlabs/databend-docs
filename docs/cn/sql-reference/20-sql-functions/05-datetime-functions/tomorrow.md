---
title: TOMORROW
---

返回明天的日期，与 `today() + 1` 相同。

## 语法

```sql
TOMORROW()
```

## 返回类型

`DATE`，返回 “YYYY-MM-DD” 格式的日期。

## 示例

```sql
SELECT TOMORROW(), TODAY()+1;

┌──────────────────────────┐
│ tomorrow() │ today() + 1 │
├────────────┼─────────────┤
│ 2024-05-23 │ 2024-05-23  │
└────────────┴─────────────┘
```
