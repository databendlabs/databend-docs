---
title: TO_YEAR
---

将日期或带时间的日期（时间戳/日期时间）转换为包含年份号（公元）的 UInt16 数字。

## 语法

```sql
TO_YEAR(<expr>)
```

## 参数

| 参数     | 描述         |
|----------|--------------|
| `<expr>` | 日期/时间戳 |

## 别名

- [YEAR](year.md)

## 返回类型

 `SMALLINT`

## 示例

```sql
SELECT NOW(), TO_YEAR(NOW()), YEAR(NOW());

┌───────────────────────────────────────────────────────────┐
│            now()           │ to_year(now()) │ year(now()) │
├────────────────────────────┼────────────────┼─────────────┤
│ 2024-03-14 23:37:03.895166 │           2024 │        2024 │
└───────────────────────────────────────────────────────────┘
```
