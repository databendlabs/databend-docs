---
title: TO_DAY_OF_WEEK
---

将日期或带时间的日期（时间戳/日期时间）转换为包含星期几的 UInt8 数字（星期一为 1，星期日为 7）。

## 语法

```sql
TO_DAY_OF_WEEK(<expr>)
```

## 参数

| 参数       | 描述         |
|-----------|--------------|
| `<expr>`  | 日期/时间戳  |

## 返回类型

`TINYINT`

## 示例

```sql

SELECT
    to_day_of_week('2023-11-12 09:38:18.165575');

┌──────────────────────────────────────────────┐
│ to_day_of_week('2023-11-12 09:38:18.165575') │
├──────────────────────────────────────────────┤
│                                            7 │
└──────────────────────────────────────────────┘
```
