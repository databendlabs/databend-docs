---
title: NOW
---

返回当前日期和时间。

## 语法

```sql
NOW()
```

## 返回类型

TIMESTAMP

## 别名

- [CURRENT_TIMESTAMP](current-timestamp.md)

## 示例

此示例返回当前日期和时间：

```sql
SELECT CURRENT_TIMESTAMP(), NOW();

┌─────────────────────────────────────────────────────────┐
│     current_timestamp()    │            now()           │
├────────────────────────────┼────────────────────────────┤
│ 2024-01-29 04:38:12.584359 │ 2024-01-29 04:38:12.584417 │
└─────────────────────────────────────────────────────────┘
```
