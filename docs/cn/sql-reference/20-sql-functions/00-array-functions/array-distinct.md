---
title: ARRAY_DISTINCT
---

删除数组中所有重复项和 NULL 值，但不保留原始顺序。

## 句法

```sql
ARRAY_DISTINCT( <array> )
```

## 示例

```sql
SELECT ARRAY_DISTINCT([1, 2, 2, 4, 3]);

┌─────────────────────────────────┐
│ array_distinct([1, 2, 2, 4, 3]) │
├─────────────────────────────────┤
│ [1,2,4,3]                       │
└─────────────────────────────────┘
```
