---
title: SLICE
---

从数组中按索引（从 1 开始）提取一个切片。

## 语法

```sql
SLICE( <array>, <start>[, <end>] )
```

## 别名

- [ARRAY_SLICE](array-slice.md)

## 示例

```sql
SELECT ARRAY_SLICE([1, 21, 32, 4], 2, 3), SLICE([1, 21, 32, 4], 2, 3);

┌─────────────────────────────────────────────────────────────────┐
│ array_slice([1, 21, 32, 4], 2, 3) │ slice([1, 21, 32, 4], 2, 3) │
├───────────────────────────────────┼─────────────────────────────┤
│ [21,32]                           │ [21,32]                     │
└─────────────────────────────────────────────────────────────────┘
```
