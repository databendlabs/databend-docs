---
title: ARRAY_FILTER
---

从输入数组中构建一个新数组，该数组仅包含使 lambda 函数返回 true 的元素。

## 语法

```sql
ARRAY_FILTER( <array>, <lambda> )
```

## 示例

```sql
SELECT ARRAY_FILTER([1, 2, 3], x -> x > 1);

┌───────────────────────────────────────┐
│ array_filter([1, 2, 3], x -> (x > 1)) │
├───────────────────────────────────────┤
│ [2,3]                                 │
└───────────────────────────────────────┘
```
