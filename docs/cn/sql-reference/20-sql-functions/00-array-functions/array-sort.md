---
title: ARRAY_SORT
---

对数组中的元素进行升序排序。

## 语法

```sql
ARRAY_SORT( <array>[, <order>, <nullposition>] )
```

| 参数          | 默认值      | 描述                                                                                 |
|---------------|-------------|------------------------------------------------------------------------------------|
| order         | ASC         | 指定排序顺序为升序（ASC）或降序（DESC）。                                           |
| nullposition  | NULLS FIRST | 确定排序结果中NULL值的位置，是在开始（NULLS FIRST）还是在结束（NULLS LAST）处。     |

## 示例

```sql
SELECT ARRAY_SORT([1, 4, 3, 2]);

┌──────────────────────────┐
│ array_sort([1, 4, 3, 2]) │
├──────────────────────────┤
│ [1,2,3,4]                │
└──────────────────────────┘
```
