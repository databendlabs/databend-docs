---
title: GET
---

根据索引（从 1 开始）从数组中返回一个元素。

## 语法

```sql
GET( <array>, <index> )
```

## 别名

- [ARRAY_GET](array-get.md)

## 示例

```sql
SELECT GET([1, 2], 2), ARRAY_GET([1, 2], 2);

┌───────────────────────────────────────┐
│ get([1, 2], 2) │ array_get([1, 2], 2) │
├────────────────┼──────────────────────┤
│              2 │                    2 │
└───────────────────────────────────────┘
```
