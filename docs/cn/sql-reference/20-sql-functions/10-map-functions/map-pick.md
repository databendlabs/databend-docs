返回一个新的 MAP，其中包含现有 MAP 中的指定键值对。

## 语法

```sql
MAP_PICK( <map>, <key1> [, <key2>, ... ] )
MAP_PICK( <map>, <array> )
```

## 参数

| 参数 | 描述 |
|-----------|-------------------------------------------------------- |
| `<map>`   | 输入 MAP。                                          |
| `<keyN>`  | 要从返回的 MAP 中包含的 KEY。          |
| `<array>` | 要从返回的 MAP 中包含的 KEY 数组。 |

:::note
- 键表达式的类型和 map 中的键必须相同。
- 将忽略在 map 中找不到的键值。
:::

## 返回类型

Map。

## 示例

```sql
SELECT MAP_PICK({'a':1,'b':2,'c':3}, 'a', 'c');
┌─────────────────────────────────────────┐
│ map_pick({'a':1,'b':2,'c':3}, 'a', 'c') │
├─────────────────────────────────────────┤
│ {'a':1,'c':3}                           │
└─────────────────────────────────────────┘

SELECT MAP_PICK({'a':1,'b':2,'c':3}, ['a', 'b']);
┌───────────────────────────────────────────┐
│ map_pick({'a':1,'b':2,'c':3}, ['a', 'b']) │
├───────────────────────────────────────────┤
│ {'a':1,'b':2}                             │
└───────────────────────────────────────────┘
```
