---
title: POW
---

返回 `x` 的 `y` 次方。

## 语法

```sql
POW( <x, y> )
```

## 别名

- [POWER](power.md)

## 示例

```sql
SELECT POW(-2, 2), POWER(-2, 2);

┌─────────────────────────────────┐
│ pow((- 2), 2) │ power((- 2), 2) │
├───────────────┼─────────────────┤
│             4 │               4 │
└─────────────────────────────────┘
```
