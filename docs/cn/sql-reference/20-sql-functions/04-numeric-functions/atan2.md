---
title: ATAN2
---

返回两个参数 `x` 和 `y` 的反正切值。其原理类似于计算 `y` / `x` 的反正切，但会同时考虑两个参数的符号以确定结果所在象限。`ATAN(y, x)` 是 `ATAN2(y, x)` 的同义词。

## 语法

```sql
ATAN2( <y, x> )
```

## 示例

```sql
SELECT ATAN2(-2, 2);

┌─────────────────────┐
│   atan2((- 2), 2)   │
├─────────────────────┤
│ -0.7853981633974483 │
└─────────────────────┘
```
