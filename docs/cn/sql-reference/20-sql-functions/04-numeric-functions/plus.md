---
title: PLUS
---

计算两个数值或小数值的总和。

## 语法

```sql
PLUS(<number1>, <number2>)
```

## 别名

- [ADD](add.md)

## 示例

```sql
SELECT ADD(1, 2.3), PLUS(1, 2.3);

┌───────────────────────────────┐
│  add(1, 2.3)  │  plus(1, 2.3) │
├───────────────┼───────────────┤
│ 3.3           │ 3.3           │
└───────────────────────────────┘
```
