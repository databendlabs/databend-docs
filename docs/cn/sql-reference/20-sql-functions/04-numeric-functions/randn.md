---
title: RAND(n)
---

返回在 `0 <= v < 1.0` 范围内的随机浮点值 v。要获取 `i <= R < j` 范围内的随机整数 R，请使用表达式 `FLOOR(i + RAND() * (j − i))`。参数 `n` 作为种子值使用。当参数值相同时，RAND(n) 每次都会返回相同结果，从而生成可重复的列值序列。

## 语法

```sql
RAND( <n>)
```

## 示例

```sql
SELECT RAND(1);

┌────────────────────┐
│       rand(1)      │
├────────────────────┤
│ 0.7133693869548766 │
└────────────────────┘
```
