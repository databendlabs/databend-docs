---
title: RAND(n)
---

返回一个随机浮点数值 v，范围在 `0 <= v < 1.0` 之间。要获得范围在 `i <= R < j` 之间的随机整数 R，请使用表达式 `FLOOR(i + RAND() * (j − i))`。参数 `n` 用作种子值。对于相等的参数值，RAND(n) 每次都返回相同的值，从而产生可重复的列值序列。

## 语法

```sql
RAND( <n>)
```

## 示例

```sql
SELECT RAND(1);

┌────────────────────┐
│       rand(1)      │
├────────────────────┤
│ 0.7133693869548766 │
└────────────────────┘
```
