返回第一个数除以第二个数的商。如果第二个数为0，则返回0。

另请参阅：

- [DIV](div.md)
- [DIVNULL](divnull.md)

## 语法

```sql
DIV0(<number1>, <number2>)
```

## 示例

```sql
SELECT
  DIV0(20, 6),
  DIV0(20, 0),
  DIV0(20, NULL);

┌───────────────────────────────────────────────────┐
│     div0(20, 6)    │ div0(20, 0) │ div0(20, null) │
├────────────────────┼─────────────┼────────────────┤
│ 3.3333333333333335 │           0 │ NULL           │
└───────────────────────────────────────────────────┘
```
