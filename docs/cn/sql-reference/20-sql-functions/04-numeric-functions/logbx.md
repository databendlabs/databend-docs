---
title: "LOG(b, x)"
---

返回以 b 为底数的 `x` 的对数。若 `x` 小于或等于 0.0E0，则函数返回 NULL。

## 语法

```sql
LOG( <b, x> )
```

## 示例

```sql
SELECT LOG(2, 65536);

┌───────────────┐
│ log(2, 65536) │
├───────────────┤
│            16 │
└───────────────┘
```
