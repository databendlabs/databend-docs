---
title: RAND()
---

返回一个在 `0 <= v < 1.0` 范围内的随机浮点数 `v`。要获取一个在 `i <= R < j` 范围内的随机整数 `R`，可以使用表达式 `FLOOR(i + RAND() * (j − i))`。

## 语法

```sql
RAND()
```

## 示例

```sql
SELECT RAND();

┌────────────────────┐
│       rand()       │
├────────────────────┤
│ 0.5191511074382174 │
└────────────────────┘
```
