---
title: RAND()
---

返回一个范围在 `0 <= v < 1.0` 内的随机浮点值 v。要获取范围在 `i <= R < j` 内的随机整数 R，可以使用表达式 `FLOOR(i + RAND() * (j − i))`。

## 语法

```sql
RAND()
```

## 示例

```sql
SELECT RAND();

┌────────────────────┐
│       rand()       │
├────────────────────┤
│ 0.5191511074382174 │
└────────────────────┘
```
