---
title: RAND()
---

返回一个随机浮点数值 v，范围在 `0 <= v < 1.0` 之间。要获得范围在 `i <= R < j` 之间的随机整数 R，请使用表达式 `FLOOR(i + RAND() * (j − i))`。

## 语法

```sql
RAND()
```

## 示例

```sql
SELECT RAND();

┌────────────────────┐
│       rand()       │
├────────────────────┤
│ 0.5191511074382174 │
└────────────────────┘
```
