---
title: MULTIPLY
---

MULTIPLY 函数用于两个数字的乘法运算，等价于 `*` 运算符。

## 语法

```sql
MULTIPLY(x, y)
-- 或使用运算符语法
x * y
```

## 参数

| 参数      | 描述                 |
|-----------|----------------------|
| x, y      | 要相乘的数值表达式。 |

## 返回类型

根据输入参数返回对应数据类型的数值。

## 示例

```sql
-- 使用函数语法
SELECT MULTIPLY(5, 3);
+----------------+
| MULTIPLY(5, 3) |
+----------------+
|             15 |
+----------------+

-- 使用运算符语法
SELECT 5 * 3;
+-------+
| 5 * 3 |
+-------+
|    15 |
+-------+

-- 使用小数
SELECT MULTIPLY(2.5, 4);
+------------------+
| MULTIPLY(2.5, 4) |
+------------------+
|             10.0 |
+------------------+

-- 使用列引用
SELECT number, MULTIPLY(number, 10) AS multiplied 
FROM numbers(5);
+--------+------------+
| number | multiplied |
+--------+------------+
|      0 |          0 |
|      1 |         10 |
|      2 |         20 |
|      3 |         30 |
|      4 |         40 |
+--------+------------+
```
