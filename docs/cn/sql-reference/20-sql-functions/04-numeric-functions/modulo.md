---
title: MODULO
---

返回 `x` 除以 `y` 的余数。如果 `y` 为 0，则返回错误。

## 语法

```sql
MODULO( <x>, <y> )
```

## 别名

- [MOD](mod.md)

## 示例

```sql
SELECT MOD(9, 2), MODULO(9, 2);

┌──────────────────────────┐
│ mod(9, 2) │ modulo(9, 2) │
├───────────┼──────────────┤
│         1 │            1 │
└──────────────────────────┘
```
