---
title: BITMAP_MIN
---

获取位图中包含的最小值。

## 语法

```sql
BITMAP_MIN( <bitmap> )
```

## 示例

```sql
SELECT BITMAP_MIN(BUILD_BITMAP([1,4,5]));

┌─────────────────────────────────────┐
│ bitmap_min(build_bitmap([1, 4, 5])) │
├─────────────────────────────────────┤
│                                   1 │
└─────────────────────────────────────┘
```
