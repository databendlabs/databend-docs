使用“等高”分桶策略生成数据分布直方图。

## 语法

```sql
HISTOGRAM(<expr>)

-- 以下两种形式是等效的：
HISTOGRAM(<max_num_buckets>)(<expr>)
HISTOGRAM(<expr> [, <max_num_buckets>])
```

| 参数              | 描述                                                                            |
|-------------------|---------------------------------------------------------------------------------|
| `expr`            | `expr` 的数据类型应该是可排序的。                                                    |
| `max_num_buckets` | 可选的正整数，指定最大桶数。默认为 128。                                                |

## 返回类型

返回一个空字符串或一个具有以下结构的 JSON 对象：

- **buckets**: 包含详细信息的桶列表：
  - **lower**: 桶的下限。
  - **upper**: 桶的上限。
  - **count**: 桶中的元素数量。
  - **pre_sum**: 截至当前桶的元素累计计数。
  - **ndv**: 桶中不同值的数量。

## 示例

此示例显示了 HISTOGRAM 函数如何分析 `histagg` 表中 `c_int` 值的分布，返回桶边界、不同值的计数、元素计数和累计计数：

```sql
CREATE TABLE histagg (
  c_id INT,
  c_tinyint TINYINT,
  c_smallint SMALLINT,
  c_int INT
);

INSERT INTO histagg VALUES
  (1, 10, 20, 30),
  (1, 11, 21, 33),
  (1, 11, 12, 13),
  (2, 21, 22, 23),
  (2, 31, 32, 33),
  (2, 10, 20, 30);

SELECT HISTOGRAM(c_int) FROM histagg;

```
