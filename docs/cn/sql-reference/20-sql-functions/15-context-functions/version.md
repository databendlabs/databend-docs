---
title: VERSION
---

Returns the current version of DatabendQuery.

## Syntax

```sql
VERSION()
```

## Examples

```sql
SELECT VERSION();

┌───────────────────────────────────────────────────────────────────────────────────────────────┐
│                                           version()                                           │
├───────────────────────────────────────────────────────────────────────────────────────────────┤
│ DatabendQuery v1.2.252-nightly-193ed56304(rust-1.75.0-nightly-2023-12-12T22:07:25.371440000Z) │
└───────────────────────────────────────────────────────────────────────────────────────────────┘
```
