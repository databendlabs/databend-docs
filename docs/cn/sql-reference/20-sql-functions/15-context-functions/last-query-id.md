---
title: LAST_QUERY_ID
---

返回当前会话中最近查询的查询ID，索引可以是(-1, 1, 1+2)...，超出范围的索引将返回空字符串。

## 语法

```sql
LAST_QUERY_ID(<index>)
```

## 示例

```sql
SELECT LAST_QUERY_ID(-1);

┌──────────────────────────────────────┐
│         last_query_id((- 1))         │
├──────────────────────────────────────┤
│ a6f615c6-5bad-4863-8558-afd01889448c │
└──────────────────────────────────────┘
```
