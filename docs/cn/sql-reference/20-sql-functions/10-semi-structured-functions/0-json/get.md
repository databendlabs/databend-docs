---
title: GET
---

从包含 `数组（ARRAY）` 的 `变体（Variant）` 中按 `index` 提取值，或从包含 `对象（OBJECT）` 的 `变体（Variant）` 中按 `field_name` 提取值。如果任一参数为 `NULL`，则返回 `NULL`；否则返回提取的值（类型为 `变体（Variant）`）。

`GET` 对 `field_name` 执行区分大小写的匹配。如需不区分大小写的匹配，请使用 `GET_IGNORE_CASE`。

## 语法

```sql
GET( <variant>, <index> )

GET( <variant>, <field_name> )
```

## 参数

| 参数 | 说明 |
|----------------|------------------------------------------------------------------|
| `<variant>` | 包含 `数组（ARRAY）` 或 `对象（OBJECT）` 的 `变体（VARIANT）` 值 |
| `<index>` | Uint32 类型的值，用于指定值在 `数组（ARRAY）` 中的位置 |
| `<field_name>` | 字符串（String）类型的值，用于指定 `对象（OBJECT）` 键值对中的键 |

## 返回类型

变体（VARIANT）

## 示例

```sql
SELECT get(parse_json('[2.71, 3.14]'), 0);
+------------------------------------+
| get(parse_json('[2.71, 3.14]'), 0) |
+------------------------------------+
| 2.71                               |
+------------------------------------+

SELECT get(parse_json('{"aa":1, "aA":2, "Aa":3}'), 'aa');
+---------------------------------------------------+
| get(parse_json('{"aa":1, "aA":2, "Aa":3}'), 'aa') |
+---------------------------------------------------+
| 1                                                 |
+---------------------------------------------------+

SELECT get(parse_json('{"aa":1, "aA":2, "Aa":3}'), 'AA');
+---------------------------------------------------+
| get(parse_json('{"aa":1, "aA":2, "Aa":3}'), 'AA') |
+---------------------------------------------------+
| NULL                                              |
+---------------------------------------------------+
```
