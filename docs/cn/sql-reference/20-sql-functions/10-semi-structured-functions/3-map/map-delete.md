返回一个移除了一个或多个键的现有 MAP（映射）。

## 语法

```sql
MAP_DELETE( <map>, <key1> [, <key2>, ... ] )
MAP_DELETE( <map>, <array> )
```

## 参数

| 参数 | 描述 |
|-----------|--------------------------------------------------------|
| `<map>`   | 包含要移除的键（KEY）的 MAP（映射）。 |
| `<keyN>`  | 从返回的 MAP（映射）中移除的键（KEY）。 |
| `<array>` | 从返回的 MAP（映射）中移除的键（KEY）的数组（Array）。 |

:::note
- 键表达式的类型必须与 MAP（映射）中的键的类型相同。
- 在 MAP（映射）中未找到的键值将被忽略。
:::

## 返回类型

Map。

## 示例

```sql
SELECT MAP_DELETE({'a':1,'b':2,'c':3}, 'a', 'c');
┌───────────────────────────────────────────┐
│ map_delete({'a':1,'b':2,'c':3}, 'a', 'c') │
├───────────────────────────────────────────┤
│ {'b':2}                                   │
└───────────────────────────────────────────┘

SELECT MAP_DELETE({'a':1,'b':2,'c':3}, ['a', 'b']);
┌─────────────────────────────────────────────┐
│ map_delete({'a':1,'b':2,'c':3}, ['a', 'b']) │
├─────────────────────────────────────────────┤
│ {'c':3}                                     │
└─────────────────────────────────────────────┘
```
