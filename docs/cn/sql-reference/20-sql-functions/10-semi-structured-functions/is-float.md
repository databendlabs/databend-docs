检查输入的 JSON 值是否为浮点数。

## 语法

```sql
IS_FLOAT( <expr> )
```

## 返回类型

如果输入的 JSON 值是浮点数，则返回 `true`，否则返回 `false`。

## 示例

```sql
SELECT
  IS_FLOAT(PARSE_JSON('1.23')),
  IS_FLOAT(PARSE_JSON('[1,2,3]'));

┌────────────────────────────────────────────────────────────────┐
│ is_float(parse_json('1.23')) │ is_float(parse_json('[1,2,3]')) │
├──────────────────────────────┼─────────────────────────────────┤
│ true                         │ false                           │
└────────────────────────────────────────────────────────────────┘
```
