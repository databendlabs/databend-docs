title: 序列函数（Sequence Functions）

本节提供 Databend 中序列函数的参考信息。序列函数允许您使用序列对象（Sequence Objects），这些对象生成唯一的、自动递增的数值。

## 可用的序列函数

| 函数 | 描述 | 示例 |
|------|------|------|
| [NEXTVAL](nextval) | 从序列中检索下一个值 | `NEXTVAL(my_sequence)` |
