---
title: IFNULL
---

如果 `<expr1>` 为 NULL，则返回 `<expr2>`，否则返回 `<expr1>`。

## 语法

```sql
IFNULL(<expr1>, <expr2>)
```

## 别名

- [NVL](nvl.md)

## 示例

```sql
SELECT IFNULL(NULL, 'b'), IFNULL('a', 'b');

┌──────────────────────────────────────┐
│ ifnull(null, 'b') │ ifnull('a', 'b') │
├───────────────────┼──────────────────┤
│ b                 │ a                │
└──────────────────────────────────────┘

SELECT IFNULL(NULL, 2), IFNULL(1, 2);

┌────────────────────────────────┐
│ ifnull(null, 2) │ ifnull(1, 2) │
├─────────────────┼──────────────┤
│               2 │            1 │
└────────────────────────────────┘
```
