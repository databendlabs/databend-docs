---
title: GEN_RANDOM_UUID
---

Generates a random UUID based on v4.

## Syntax

```sql
GEN_RANDOM_UUID()
```

## Examples

```sql
SELECT GEN_RANDOM_UUID();

┌──────────────────────────────────────┐
│           gen_random_uuid()          │
├──────────────────────────────────────┤
│ 77c4a531-ffb4-4a35-9af6-832a3d2abc77 │
└──────────────────────────────────────┘
```
