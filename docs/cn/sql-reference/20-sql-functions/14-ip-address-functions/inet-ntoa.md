---
title: INET_NTOA
---

将一个32位整数转换为IPv4地址。

## 语法

```sql
INET_NOTA( <int32> )
```

## 别名

- [IPV4_NUM_TO_STRING](ipv4-num-to-string.md)

## 返回类型

字符串。

## 示例

```sql
SELECT IPV4_NUM_TO_STRING(16909060), INET_NTOA(16909060);

┌────────────────────────────────────────────────────┐
│ ipv4_num_to_string(16909060) │ inet_ntoa(16909060) │
├──────────────────────────────┼─────────────────────┤
│ 1.2.3.4                      │ 1.2.3.4             │
└────────────────────────────────────────────────────┘
```
