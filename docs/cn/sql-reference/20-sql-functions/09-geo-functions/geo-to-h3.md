---
title: GEO_TO_H3
---

返回给定位置所在的六边形单元格的 [H3](https://eng.uber.com/h3/) 索引。返回 0 表示发生了错误。

## 语法

```sql
GEO_TO_H3(lon, lat, res)
```

## 示例

```sql
SELECT GEO_TO_H3(37.79506683, 55.71290588, 15);

┌─────────────────────────────────────────┐
│ geo_to_h3(37.79506683, 55.71290588, 15) │
├─────────────────────────────────────────┤
│                      644325524701193974 │
└─────────────────────────────────────────┘
```
