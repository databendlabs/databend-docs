---
title: H3_EDGE_LENGTH_KM
---

返回给定分辨率下六边形边的平均长度，单位为公里。不包括五边形。

## 语法

```sql
H3_EDGE_LENGTH_KM(res)
```

## 示例

```sql
SELECT H3_EDGE_LENGTH_KM(1);

┌──────────────────────┐
│ h3_edge_length_km(1) │
├──────────────────────┤
│    483.0568390711111 │
└──────────────────────┘
```
