---
title: H3_GET_UNIDIRECTIONAL_EDGES_FROM_HEXAGON
---

返回从提供的 H3Index 中获取的所有单向边。

## 语法

```sql
H3_GET_UNIDIRECTIONAL_EDGES_FROM_HEXAGON(h3)
```

## 示例

```sql
SELECT H3_GET_UNIDIRECTIONAL_EDGES_FROM_HEXAGON(644325524701193754);
```
