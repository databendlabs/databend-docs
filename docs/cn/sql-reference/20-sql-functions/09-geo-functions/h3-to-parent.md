---
title: H3_TO_PARENT
---

返回在 `parent_res` 分辨率下包含 `h3` 的父级索引。返回 0 表示发生错误。

## 语法

```sql
H3_TO_PARENT(h3, parent_res)
```

## 示例

```sql
SELECT H3_TO_PARENT(635318325446452991, 12);

┌──────────────────────────────────────┐
│ h3_to_parent(635318325446452991, 12) │
├──────────────────────────────────────┤
│                   630814725819082751 │
└──────────────────────────────────────┘
```
