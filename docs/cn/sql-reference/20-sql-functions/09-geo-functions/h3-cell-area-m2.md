---
title: H3_CELL_AREA_M2
---

返回特定单元格的精确面积，单位为平方米。

## 语法

```sql
H3_CELL_AREA_M2(h3)
```

## 示例

```sql
SELECT H3_CELL_AREA_M2(599119489002373119);

┌─────────────────────────────────────┐
│ h3_cell_area_m2(599119489002373119) │
├─────────────────────────────────────┤
│                  127785582.60809991 │
└─────────────────────────────────────┘
```
