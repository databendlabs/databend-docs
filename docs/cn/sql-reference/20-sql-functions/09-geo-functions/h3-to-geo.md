---
title: H3_TO_GEO
---

返回与给定 [H3](https://eng.uber.com/h3/) 索引相对应的经度和纬度。

## 语法

```sql
H3_TO_GEO(h3)
```

## 示例

```sql
SELECT H3_TO_GEO(644325524701193974);

┌────────────────────────────────────────┐
│      h3_to_geo(644325524701193974)     │
├────────────────────────────────────────┤
│ (37.79506616830255,55.712902431456676) │
└────────────────────────────────────────┘
```
