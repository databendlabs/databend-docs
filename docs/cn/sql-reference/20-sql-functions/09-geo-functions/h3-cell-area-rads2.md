---
title: H3_CELL_AREA_RADS2
---

返回特定单元格的精确面积，单位为平方弧度。

## 语法

```sql
H3_CELL_AREA_RADS2(h3)
```

## 示例

```sql
SELECT H3_CELL_AREA_RADS2(599119489002373119);

┌────────────────────────────────────────┐
│ h3_cell_area_rads2(599119489002373119) │
├────────────────────────────────────────┤
│                0.000003148224310427697 │
└────────────────────────────────────────┘
```
