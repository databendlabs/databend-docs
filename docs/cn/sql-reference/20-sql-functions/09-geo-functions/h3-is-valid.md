---
title: H3_IS_VALID
---

检查给定的 [H3](https://eng.uber.com/h3/) 索引是否有效。

## 语法

```sql
H3_IS_VALID(h3)
```

## 示例

```sql
SELECT H3_IS_VALID(644325524701193974);

┌─────────────────────────────────┐
│ h3_is_valid(644325524701193974) │
├─────────────────────────────────┤
│ true                            │
└─────────────────────────────────┘
```
