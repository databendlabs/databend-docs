---
title: H3_TO_STRING
---

将给定的 [H3](https://eng.uber.com/h3/) 索引表示形式转换为字符串表示形式。

## 语法

```sql
H3_TO_STRING(h3)
```

## 示例

```sql
SELECT H3_TO_STRING(635318325446452991);

┌──────────────────────────────────┐
│ h3_to_string(635318325446452991) │
├──────────────────────────────────┤
│ 8d11aa6a38826ff                  │
└──────────────────────────────────┘
```
