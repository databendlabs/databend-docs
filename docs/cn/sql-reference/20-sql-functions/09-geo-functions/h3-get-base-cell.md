---
title: H3_GET_BASE_CELL
---

返回给定 [H3](https://eng.uber.com/h3/) 索引的基本单元格编号。

## 语法

```sql
H3_GET_BASE_CELL(h3)
```

## 示例

```sql
SELECT H3_GET_BASE_CELL(644325524701193974);

┌──────────────────────────────────────┐
│ h3_get_base_cell(644325524701193974) │
├──────────────────────────────────────┤
│                                    8 │
└──────────────────────────────────────┘
```
