---
title: H3_IS_PENTAGON
---

检查给定的 [H3](https://eng.uber.com/h3/) 索引是否表示一个五边形单元格。

## 语法

```sql
H3_IS_PENTAGON(h3)
```

## 示例

```sql
SELECT H3_IS_PENTAGON(599119489002373119);

┌────────────────────────────────────┐
│ h3_is_pentagon(599119489002373119) │
├────────────────────────────────────┤
│ true                               │
└────────────────────────────────────┘
```
