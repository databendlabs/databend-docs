---
title: STRCMP
---

如果字符串相同则返回0，如果第一个参数小于第二个参数则返回-1，否则返回1。

## 语法

```sql
STRCMP(<expr1> ,<expr2>)
```

## 参数

| 参数       | 描述       |
|-----------|------------|
| `<expr1>` | 字符串     |
| `<expr2>` | 字符串     |

## 返回类型

`BIGINT`

## 示例

```sql
SELECT STRCMP('text', 'text2');
+-------------------------+
| STRCMP('text', 'text2') |
+-------------------------+
|                      -1 |
+-------------------------+

SELECT STRCMP('text2', 'text');
+-------------------------+
| STRCMP('text2', 'text') |
+-------------------------+
|                       1 |
+-------------------------+

SELECT STRCMP('text', 'text');
+------------------------+
| STRCMP('text', 'text') |
+------------------------+
|                      0 |
+------------------------+
```
