---
title: ORD
---

如果最左边的字符不是多字节字符，则 ORD() 返回的值与 ASCII() 函数相同。

如果字符串 str 的最左边的字符是多字节字符，则返回该字符的代码，
该代码通过使用以下公式从其组成字节的数值计算得出：

```sql
  (第 1 个字节代码)
+ (第 2 个字节代码 * 256)
+ (第 3 个字节代码 * 256^2) ...
```

## 语法

```sql
ORD(<str>)
```

## 参数

| 参数 | 描述 |
|-----------|-------------|
| `<str>`   | 字符串。 |

## 返回类型

`BIGINT`

## 示例

```sql
SELECT ORD('2')
+--------+
| ORD(2) |
+--------+
|     50 |
+--------+
```
