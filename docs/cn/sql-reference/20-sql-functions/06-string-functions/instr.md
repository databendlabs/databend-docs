---
title: INSTR
---

返回字符串 `str` 中子字符串 `substr` 第一次出现的位置。
这与 `LOCATE()` 的双参数形式相同，只是参数的顺序相反。

## 语法

```sql
INSTR(<str>, <substr>)
```

## 参数

| 参数       | 描述         |
|------------|--------------|
| `<str>`    | 字符串。     |
| `<substr>` | 子字符串。   |

## 返回类型

`BIGINT`

## 示例

```sql
SELECT INSTR('foobarbar', 'bar');
+---------------------------+
| INSTR('foobarbar', 'bar') |
+---------------------------+
|                         4 |
+---------------------------+

SELECT INSTR('xbar', 'foobar');
+-------------------------+
| INSTR('xbar', 'foobar') |
+-------------------------+
|                       0 |
+-------------------------+
```
