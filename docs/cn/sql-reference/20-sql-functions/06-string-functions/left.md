---
title: LEFT
---

返回字符串 `str` 最左边的 `len` 个字符，如果任一参数为 NULL，则返回 NULL。如果 `len` 大于 `str` 的长度，则返回整个 `str`。

## 语法

```sql
LEFT(<str>, <len>);
```

## 参数

| 参数     | 描述                                                  |
|----------|------------------------------------------------------|
| `<str>`  | 要从中提取字符的主字符串                              |
| `<len>`  | 要提取的字符数量                                      |

## 返回类型

`VARCHAR`

## 示例

```sql
SELECT LEFT('foobarbar', 5), LEFT('foobarbar', 10);

┌──────────────────────────────────────────────┐
│ left('foobarbar', 5) │ left('foobarbar', 10) │
├──────────────────────┼───────────────────────┤
│ fooba                │ foobarbar             │
└──────────────────────────────────────────────┘
```
