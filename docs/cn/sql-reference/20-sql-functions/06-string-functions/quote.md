---
title: QUOTE
---

引用字符串以生成一个结果，该结果可以用作 SQL 语句中正确转义的数据值。

## 语法

```sql
QUOTE(<str>)
```

## 示例

```sql
SELECT QUOTE('Don\'t!');
+-----------------+
| QUOTE('Don't!') |
+-----------------+
| Don\'t!         |
+-----------------+

SELECT QUOTE(NULL);
+-------------+
| QUOTE(NULL) |
+-------------+
|        NULL |
+-------------+
```
