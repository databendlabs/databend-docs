---
title: QUOTE
---

将字符串引用，以生成一个可以在 SQL 语句中作为正确转义的数据值使用的结果。

## 语法

```sql
QUOTE(<str>)
```

## 示例

```sql
SELECT QUOTE('Don\'t!');
+-----------------+
| QUOTE('Don't!') |
+-----------------+
| Don\'t!         |
+-----------------+

SELECT QUOTE(NULL);
+-------------+
| QUOTE(NULL) |
+-------------+
|        NULL |
+-------------+
```
