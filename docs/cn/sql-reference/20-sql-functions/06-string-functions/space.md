---
title: SPACE
---

返回一个由 N 个空格字符组成的字符串。

## 句法

```sql
SPACE(<n>);
```

## 参数

| 参数 | 描述          |
|-----------|----------------------|
| `<n>`     | 空格的数量 |

## 返回类型

字符串数据类型值。

## 示例

```sql
SELECT SPACE(20)
+----------------------+
| SPACE(20)            |
+----------------------+
|                      |
+----------------------+
```
