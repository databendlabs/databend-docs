---
title: RPAD
---

返回字符串 `str`，右侧填充字符串 `padstr` 到长度为 `len` 个字符。如果 `str` 比 `len` 长，则返回值被缩短为 `len` 个字符。

## 语法

```sql
RPAD(<str>, <len>, <padstr>)
```

## 参数

| 参数       | 描述         |
|------------|--------------|
| `<str>`    | 字符串。     |
| `<len>`    | 长度。       |
| `<padstr>` | 填充字符串。 |

## 返回类型

`VARCHAR`

## 示例

```sql
SELECT RPAD('hi',5,'?');
+--------------------+
| RPAD('hi', 5, '?') |
+--------------------+
| hi???              |
+--------------------+

SELECT RPAD('hi',1,'?');
+--------------------+
| RPAD('hi', 1, '?') |
+--------------------+
| h                  |
+--------------------+
```
