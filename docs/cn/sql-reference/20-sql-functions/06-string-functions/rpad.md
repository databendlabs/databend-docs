---
title: RPAD
---

返回字符串 str，使用字符串 padstr 从右侧填充到 len 个字符的长度。
如果 str 比 len 长，则返回值会缩短为 len 个字符。

## 语法

```sql
RPAD(<str>, <len>, <padstr>)
```

## 参数

| 参数     | 描述         |
|----------|--------------|
| `<str>`  | 字符串。     |
| `<len>`  | 长度。       |
| `<padstr>` | 填充字符串。 |

## 返回类型

`VARCHAR`

## 示例

```sql
SELECT RPAD('hi',5,'?');
+--------------------+
| RPAD('hi', 5, '?') |
+--------------------+
| hi???              |
+--------------------+

SELECT RPAD('hi',1,'?');
+--------------------+
| RPAD('hi', 1, '?') |
+--------------------+
| h                  |
+--------------------+
```
