---
title: INSERT
---

返回字符串 str，其中从位置 pos 开始且长度为 len 的子字符串被字符串 newstr 替换。如果 pos 不在字符串长度范围内，则返回原始字符串。如果 len 不在字符串剩余长度范围内，则从位置 pos 替换字符串的其余部分。如果任何参数为 NULL，则返回 NULL。

## Syntax

```sql
INSERT(<str>, <pos>, <len>, <newstr>)
```

## Arguments

| Arguments  | Description     |
|------------|-----------------|
| `<str>`    | 字符串。        |
| `<pos>`    | 位置。          |
| `<len>`    | 长度。          |
| `<newstr>` | 新字符串。      |

## Return Type

`VARCHAR`
