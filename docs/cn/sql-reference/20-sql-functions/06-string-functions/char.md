title: CHAR
---

返回传递的每个整数对应的字符。

## 语法

```sql
CHAR(N, ...)
```

## 参数

| 参数      | 描述         |
|-----------|--------------|
| N         | 数值列       |

## 返回类型

`BINARY`

## 示例

```sql
SELECT CHAR(77,121,83,81,76) as a, a::String;
┌────────────────────────┐
│      a     │ a::string │
│   Binary   │   String  │
├────────────┼───────────┤
│ 4D7953514C │ MySQL     │
└────────────────────────┘
```
