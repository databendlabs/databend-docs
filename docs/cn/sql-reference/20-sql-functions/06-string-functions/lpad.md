---
title: LPAD
---

返回字符串 str，使用字符串 padstr 从左侧填充到 len 字符的长度。
如果 str 的长度大于 len，则返回值将缩短为 len 个字符。

## 语法

```sql
LPAD(<str>, <len>, <padstr>)
```

## 参数

| 参数     | 描述         |
|----------|--------------|
| `<str>`  | 字符串。     |
| `<len>`  | 长度。       |
| `<padstr>` | 填充字符串。 |

## 返回类型

`VARCHAR`

## 示例

```sql
SELECT LPAD('hi',4,'??');
+---------------------+
| LPAD('hi', 4, '??') |
+---------------------+
| ??hi                |
+---------------------+

SELECT LPAD('hi',1,'??');
+---------------------+
| LPAD('hi', 1, '??') |
+---------------------+
| h                   |
+---------------------+
```
