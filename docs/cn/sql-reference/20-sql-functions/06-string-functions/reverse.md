---
title: REVERSE
---

返回字符顺序反转的字符串 `str`。

## 语法

```sql
REVERSE(<str>)
```

## 参数

| 参数      | 描述          |
|-----------|---------------|
| `<str>`   | 字符串值。    |

## 返回类型

`VARCHAR`

## 示例

```sql
SELECT REVERSE('abc');
+----------------+
| REVERSE('abc') |
+----------------+
| cba            |
+----------------+
```
