---
title: TO_BASE64
---

将字符串参数转换为 base-64 编码形式，并将结果作为字符串返回。
如果参数不是字符串，则在转换之前将其转换为字符串。
如果参数为 NULL，则结果为 NULL。

## 语法

```sql
TO_BASE64(<v>)
```

## 参数

| 参数      | 描述        |
|-----------|-------------|
| `<v>`     | 值。        |

## 返回类型

`VARCHAR`

## 示例

```sql
SELECT TO_BASE64('abc');
+------------------+
| TO_BASE64('abc') |
+------------------+
| YWJj             |
+------------------+
```
