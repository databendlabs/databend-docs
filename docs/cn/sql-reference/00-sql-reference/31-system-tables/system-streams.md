<FunctionDescription description="引入或更新于：v1.2.223"/>

提供系统中有关流（Stream）的信息。此表中的每一行对应一个流，列中包含流的模式、注释（如果有）、关联的表名、表 ID、表版本、快照位置、失效原因（如果适用）和所有者等详细信息。
