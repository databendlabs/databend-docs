---
title: SQL 符合性
---

Databend 旨在符合 SQL 标准，特别支持 ISO/IEC 9075:2011，也就是 SQL:2011。虽然这不是一个详尽的符合性声明，Databend 包含了许多 SQL 标准所要求的特性，这些特性在语法或功能上常常有细微的差异。本页面概述了 Databend 对 SQL:2011 标准的符合程度。
