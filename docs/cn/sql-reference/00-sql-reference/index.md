- **SQL 函数：** Databend 函数的简明指南，深入解析其多样化功能，助力高效的数据管理与分析。

- **存储过程与脚本：** 涵盖 SQL 脚本语言，包括变量、控制流、结果处理以及存储过程（Stored Procedures）中的动态执行。
