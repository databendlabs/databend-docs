title: system.tables

提供所有表的元数据信息。它包括表属性、创建时间、行数、数据大小等详细信息。

另请参阅：

- [information_schema.tables](information-schema-tables.md)
- [SHOW TABLES](../../10-sql-commands/00-ddl/01-table/show-tables.md)
