Databend 进程列表指示服务器内当前正在执行的线程集所执行的操作。

另请参阅: [KILL](01-kill.md)

## 语法

```sql
SHOW PROCESSLIST [LIKE '<pattern>' | WHERE <expr>] | [LIMIT <limit>]
```

## 示例

