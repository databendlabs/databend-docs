删除由 Databend 创建的临时文件，例如溢出文件。

另请参阅：[system.temp_files](../../00-sql-reference/20-system-tables/system-temp-files.md)

## 语法

```sql
VACUUM TEMPORARY FILES [ LIMIT <limit> ]
```

| 参数    | 描述                                          |
|---------|-----------------------------------------------|
| LIMIT   | 要删除的临时文件的最大数量。                   |

## 输出

返回已删除的临时文件数量。

## 示例

```sql
SELECT * FROM system.temp_files;
```
