---
title: 逻辑运算符
---

| 运算符   | 描述                               | 示例           | 结果   |
|----------|------------------------------------|------------------|--------|
| **AND**  | 匹配两个表达式 (`a` 和 `b`)       | **1 AND 1**      | TRUE   |
| **NOT**  | 不匹配表达式                         | **NOT 1**        | FALSE  |
| **OR**   | 匹配任一表达式                       | **1 OR 0**       | TRUE   |
