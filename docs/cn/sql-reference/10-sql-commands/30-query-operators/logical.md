---
title: 逻辑运算符
---

| Operator | Description                            | Example       | Result |
|----------|----------------------------------------|---------------|--------|
| **AND**  | 同时匹配两个表达式 (`a` and `b`) | **1 AND 1**   | TRUE   |
| **NOT**  | 不匹配表达式          | **NOT 1**     | FALSE  |
| **OR**   | 匹配其中一个表达式              | **1 OR 0**    | TRUE   |
