本页面提供 Databend 中查询算子 (Query Operators) 的参考信息。

## 算子类型

| 算子类型 | 描述 |
|--------------|-------------|
| **[算术](query-operators/arithmetic)** | 数学运算（+、-、*、/、%、DIV） |
| **[比较](query-operators/comparison)** | 值比较（=、!=、&lt;、&gt;、&lt;=、&gt;=、BETWEEN、IN） |
| **[逻辑](query-operators/logical)** | 布尔逻辑（AND、OR、NOT、XOR） |
| **[JSON](query-operators/json)** | JSON 数据操作（::、-&gt;、-&gt;&gt;、@&gt;、&lt;@） |
| **[集合](query-operators/set)** | 组合查询结果（UNION、INTERSECT、EXCEPT） |
| **[子查询](query-operators/subquery)** | 嵌套查询（EXISTS、IN、ANY、ALL、SOME） |
