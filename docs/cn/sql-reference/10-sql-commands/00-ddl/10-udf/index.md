## UDF 管理

| 命令 | 描述 |
|---------|-------------|
| [CREATE FUNCTION](ddl-create-function.md) | 创建新的用户定义函数 |
| [ALTER FUNCTION](ddl-alter-function.md) | 修改现有的用户定义函数 |
| [DROP FUNCTION](ddl-drop-function.md) | 删除用户定义函数 |
| [SHOW USER FUNCTIONS](ddl-show-user-functions.md) | 列出所有用户定义函数 |
