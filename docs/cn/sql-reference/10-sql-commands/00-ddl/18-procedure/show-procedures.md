返回系统中所有存储过程的列表。

## 语法

```sql
SHOW PROCEDURES
```

## 示例

```sql
SHOW PROCEDURES;

┌────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│       名称       │ 存储过程ID │                        参数                        │            注释            │       描述       │         创建时间         │
├──────────────────┼──────────────┼─────────────────────────────────────────────────────────┼──────────────────────────────┼────────────────────────┼────────────────────────────┤
│ convert_kg_to_lb │         2104 │ convert_kg_to_lb(Decimal(4, 2)) RETURN (Decimal(10, 2)) │ 将千克转换为磅 │ 用户定义的存储过程 │ 2024-11-07 04:12:25.243143 │
└────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘
```
