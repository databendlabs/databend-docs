描述一个特定流的详细信息。

## 语法

```sql
DESC|DESCRIBE STREAM [ <database_name>. ]<stream_name>
```

## 示例

```sql
DESC STREAM books_stream_2023;

┌───────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┐

├────────────────────────────┼───────────────────┼──────────┼─────────┼─────────────────────┼──────────────────┼─────────┼─────────────┼────────────────┤
│ 2023-11-29 02:38:29.588518 │ books_stream_2023 │ default  │ default │ default.books_total │ NULL             │         │ append_only │                │
└───────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘
```
