删除为表创建的虚拟列。

## 语法

```sql
DROP VIRTUAL COLUMN [ IF EXISTS ] FOR <table>
```

## 示例

此示例删除为名为'test'的表创建的虚拟列：

```sql
DROP VIRTUAL COLUMN FOR test;
```
