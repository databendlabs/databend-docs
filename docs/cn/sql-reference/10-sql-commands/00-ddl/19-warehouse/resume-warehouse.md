恢复之前暂停的计算集群。

## 语法

```sql
RESUME WAREHOUSE <warehouse_name>
```

## 示例

此示例恢复 `test_warehouse` 计算集群：

```sql
RESUME WAREHOUSE test_warehouse;
```
