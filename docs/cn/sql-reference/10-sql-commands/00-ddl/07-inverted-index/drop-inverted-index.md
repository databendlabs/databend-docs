在 Databend 中移除一个倒排索引。

## 语法

```sql
DROP INVERTED INDEX [IF EXISTS] <index> ON [<database>.]<table>
```

## 示例

```sql
-- 删除'customer_feedback'表上的'customer_feedback_idx'倒排索引
DROP INVERTED INDEX customer_feedback_idx ON customer_feedback;
```
