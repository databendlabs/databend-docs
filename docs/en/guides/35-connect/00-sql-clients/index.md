---
title: SQL Clients
---

Databend provides native support for command-line and GUI SQL clients. Choose based on your workflow:

| Client | Type | Best For |
|--------|------|----------|
| [**BendSQL**](/guides/connect/sql-clients/bendsql) | CLI | Developers, scripting, automation |
| [**DBeaver**](/guides/connect/sql-clients/jdbc) | GUI | Visual queries, data exploration |
